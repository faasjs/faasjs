# Function: idToTitle()

> **idToTitle**(`id`): `string`

Derive a human-readable title from a raw identifier.

Splits on whitespace, underscores, and hyphens, then capitalises each segment.

## Parameters

### id

`string` \| `number`

Raw identifier.

## Returns

`string`

Title-cased string derived from the identifier.
