# Type Alias: FuncRuntime

> **FuncRuntime** = `"api"` \| `"job"`

Runtime family executing a [Func](../classes/Func.md).

`api` is used by HTTP APIs created with `defineApi()`, and `job` is used by
background jobs created with `defineJob()`.
