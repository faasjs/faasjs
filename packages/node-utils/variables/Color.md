# Variable: Color

> `const` **Color**: `object`

ANSI foreground color codes used by [colorize](../functions/colorize.md) and the built-in logger output.

`Color.DEFAULT` resets the foreground color back to the terminal default.

## Type Declaration

### BLACK

> **BLACK**: `number` = `30`

### BLUE

> **BLUE**: `number` = `34`

### CYAN

> **CYAN**: `number` = `36`

### DEFAULT

> **DEFAULT**: `number` = `39`

### GRAY

> **GRAY**: `number` = `90`

### GREEN

> **GREEN**: `number` = `32`

### MAGENTA

> **MAGENTA**: `number` = `35`

### ORANGE

> **ORANGE**: `number` = `33`

### RED

> **RED**: `number` = `31`

## Example

```ts
const color = Color.RED
```
