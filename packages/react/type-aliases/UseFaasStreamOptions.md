# Type Alias: UseFaasStreamOptions

> **UseFaasStreamOptions** = `SharedUseFaasOptions`\<`Record`\<`string`, `any`\>, `string`\>

Options that customize the [useFaasStream](../functions/useFaasStream.md) request lifecycle.

Extends the shared request options so stream consumers can control params,
skip logic, debounce timing, polling, and base URL overrides the same way
[useFaas](../functions/useFaas.md) does.
