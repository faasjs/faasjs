# 06 React Client

Use `@faasjs/react` in a Vite + React app and call a FaasJS API endpoint.

在 Vite + React 项目中使用 `@faasjs/react` 调用 FaasJS API。

## What you learn / 你将学到

- How to initialize `FaasReactClient`.
- How to call an API action from React UI.
- How to mock API calls in unit tests with `setMock`.

- 如何初始化 `FaasReactClient`。
- 如何在 React UI 里调用 API action。
- 如何通过 `setMock` 编写客户端单测。

## Run / 运行

```bash
npm install
npm run test
npm run dev
```

Build and run production mode:

```bash
npm run build
npm run start
```
