# Agent Guide

Read the right guide before making changes:

- App-building or FaasJS API usage: [`skills/faasjs-best-practices/SKILL.md`](./skills/faasjs-best-practices/SKILL.md)
- Framework/monorepo development: [`CONTRIBUTING.md`](./CONTRIBUTING.md)
- Positioning, supported stacks, dependencies, abstractions, or defaults: [`contributing/target-users.md`](./contributing/target-users.md)
- Docs, generated docs, translations, navigation, or changelog: [`contributing/documentation-sync.md`](./contributing/documentation-sync.md)

Boundaries: `skills/**` is public user guidance; `contributing/**` is maintainer guidance; specs live under `skills/*/references/specs/**`.
