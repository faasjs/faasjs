# Interface: TypedPgVitestPluginOptions

@faasjs/pg-dev package.

The published surface exposes `TypedPgVitestPlugin` plus the low-level setup helper used by the
generated Vitest setup module.

## Properties

### environments?

> `optional` **environments?**: `string`[]

Restricts the plugin to the listed Vitest environments.

### projects?

> `optional` **projects?**: `string`[]

Restricts the plugin to the listed Vitest project names.
