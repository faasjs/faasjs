# Function: idToTitle()

> **idToTitle**(`id`): `string`

Convert a snake_case, kebab-case, or spaced identifier into a title-style label.

## Parameters

### id

`string` \| `number`

Identifier to convert.

## Returns

`string`

Generated label string.

## Example

```ts
idToTitle('example_id') // 'Example Id'
```
